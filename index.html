Introduction

Crescent OS is a lightweight operating system built using the COSMOS (C# Open Source Managed Operating System) framework. It provides a minimalistic environment with basic command-line utilities and a simple file system. It is a modified version of a past unavailable COSMOS project called 'VNU', which is made by the same developer as Crescent.

Commands

Crescent OS supports a variety of commands for file management, system operations, and more. Below is a list of available commands:

  • dir - List files and directories in the current directory.
  • echo [text] - Print text to the console.
  • cd [path] - Change the current directory.
  • md [path] - Create a new directory.
  • rd [path] - Remove a directory.
  • rdf [file] - Remove a file.
  • rdrf [path] - Remove a directory and it's contents recursively.
  • touch [file] - Create a new file.
  • miv - Open the MIV text editor.
  • shutdown - Shut down the system.
  • reboot - Reboot the system.
  • clear - Clear the console screen.
  • vnu-fetch - Display system information.

File System

Crescent OS uses a simple file system managed by the COSMOS VFS (Virtual File System). The root directory is located at 0:\. The file system supports basic operations such as creating, deleting, and listing files and directories.

File System Commands

  • dir - List files and directories.
  • cd [path] - Change directory.
  • md [path] - Create a directory.
  • rd [path] - Remove a directory.
  • rdf [file] - Remove a file.
  • touch [file] - Create a file.
  • rdrf [path] - Remove a directory and it's contents recursively.

MIV Editor

MIV (Minimalistic Vi) is a simple text editor included in Crescent OS. It allows users to create and edit text files directly from the command line.

MIV Commands

  • i - Enter insert mode to start typing.
  • :wq - Save and exit.
  • :q - Exit without saving.
  • :help - Display help information.

Hardware Information

Crescent OS provides basic hardware information through the vnu-fetch command. This includes details about the CPU, RAM, and disk space.

Hardware Info Example
